## О проекте
Мобильное приложение "Monolingo" создано, чтобы сделать изучение английского языка доступным и увлекательным. Учите слова, тренируйте грамматику и отслеживайте свой прогресс в удобном формате.

## Особенности проекта
- **Интерактивное обучение:** Уроки построены в игровом формате, чтобы поддерживать интерес.
- **Тренировка навыков:** Упражнения на произношение, аудирование и письмо.
- **Статистика:** Отслеживание личного прогресса и достижение ежедневных целей.
- **Удобный интерфейс:** Минималистичный дизайн, понятный любому пользователю.

## Установка
1. Скачайте установочный файл из раздела [Releases](https://github.com/N0nashi/Monolingo/releases).
2. Установите приложение на Android устройство.
3. Запустите и начните обучение.
